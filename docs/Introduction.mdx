# DataLayer Lens Component Library

A collection of React components for the DataLayer Lens browser extension.

## Atomic Design Structure

This component library follows the Atomic Design methodology, organising components into four levels:

- **Atoms**: Basic building blocks (Toggle, SearchInput, Toast, SupportLink, ColorSwatch)
- **Molecules**: Combinations of atoms (FilterPanel, EventStats, EventRow, JsonHighlight, SourceColorEditor)
- **Organisms**: Complex components (EventPanel, SettingsDrawer)
- **Views**: Full page layouts (DevToolsPanel, Popup)

## Getting Started

Browse components by category in the sidebar. Each component includes:

- Interactive controls to adjust props
- Multiple story variants showing different states
- Auto-generated documentation

## Technology Stack

- **React 19** - UI framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Zustand** - State management
- **Lucide React** - Icons

## Design Principles

1. **Dark-first design** - Optimised for DevTools environment
2. **Compact and information-dense** - Maximise data visibility
3. **Accessible** - ARIA labels and keyboard navigation
4. **Responsive** - Works in various panel sizes
