## CSS Custom Properties

All design tokens are available as CSS custom properties in `src/tokens/index.css`:

```css
:root {
  --dl-primary: #6366f1;
  --dl-secondary: #8b5cf6;
  --dl-accent: #22d3ee;
  --dl-dark: #0f172a;
  --dl-darker: #020617;
  --dl-card: #1e293b;
  --dl-border: #334155;
  --font-sans: 'Inter', system-ui, sans-serif;
  --font-mono: 'JetBrains Mono', Consolas, monospace;
}
```

## TypeScript Tokens

Import tokens in TypeScript from `src/tokens/index.ts`:

```typescript
import { colors, typography, spacing, radii } from '@/tokens';

// Usage
const primaryColor = colors.primary; // '#6366f1'
const fontSize = typography.fontSize.sm; // '0.875rem'
```
